# Credit Report Dashboard - Power BI

## Overview
Interactive Power BI dashboard for analyzing customer credit card data, loan insights, customer demographics, and income trends.

## Features
- Customer gender analysis
- Credit limit analysis
- Personal loan insights
- Customer job analysis
- Card category distribution

## Tools Used
- Power BI
